# Pilot Research

Pilot Research is an opt-in framework for scientific research, literature reviews, and experiment design installed in this project. Use the skill tool to invoke a skill when you need it.

<EXTREMELY-IMPORTANT>
**DO NOT invoke these skills for standard tasks, software engineering tasks, casual conversation, etc.** 
ONLY invoke a skill below if the user explicitly asks for research assistance, literature review, or mentions working on a paper. If the user is just chatting or coding, ignore this framework entirely.
</EXTREMELY-IMPORTANT>

## Available Skills

| Skill | Trigger Condition | Invocation |
|-------|-------------------|------------|
| `pilot-brainstorm` | Discuss, explore, or refine a research idea, plan, or experiment design | `pilot-research:pilot-brainstorm` |
| `pilot-literature` | Find, read, or summarize papers; build or update the research wiki | `pilot-research:pilot-literature` |
| `pilot-execute` | Execute tasks from the backlog, run experiments, or manage the research process | `pilot-research:pilot-execute` |
| `pilot-write-paper` | Draft, outline, or revise a research paper | `pilot-research:pilot-write-paper` |
| `pilot-peer-review` | Get feedback on research plan, literature review, experiment design, or paper draft | `pilot-research:pilot-peer-review` |
