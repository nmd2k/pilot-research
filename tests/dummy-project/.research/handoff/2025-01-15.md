## Issues Discovered

- Mistral-7B produces non-numeric outputs in ~15% of zero-shot responses, breaking the answer parser
- OpenAI API rate limits slowed evaluation; recommend batching requests in future runs
- MATH dataset answer format requires special LaTeX parsing; not yet implemented in eval script

## Key Decisions Made

- Use zero-shot as the baseline strategy before testing CoT — isolates the effect of prompting
- Include Mistral-7B despite low performance — important for demonstrating scale-dependent CoT emergence
- Stratified difficulty labels for GSM8K will be generated by classifying problems by step count, not by the dataset's built-in difficulty (GSM8K has none)
